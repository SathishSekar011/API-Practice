Dictionary:
============
	--> used to store data values in key:value pairs
	--> ordered, changeable and do not allow duplicates
	--> written with curly brackets and have keys and values


Syntax:
=======
	dictname = {
		key1:Value1
		key2:Value2
	}
		(or)
	dictname = dict(key1=Value1,key2=Value2) 

Accessing Items:
-----------------
	dictname[key]


Methods in Dictionary:
=======================
--> get(key)	--> to retain a value
--> keys() 	--> return a list of all the keys in the dictionary
--> values() 	--> return a list of all the values in the dictionary
--> items() 	--> return each item in a dictionary as tuples in a list
