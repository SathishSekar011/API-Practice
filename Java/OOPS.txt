OOPS:
=====
	--> Object Oriented Programming Structure
	--> programme must be a collection of classes , methods , object

Object	--> memmory needed for your programme to execute a method
Method	--> action performing inside a class
Class	--> Collections of Methods and Objects
Package --> Collections of Classes
Project --> Collections of Packages

--> Inheritance
--> Polymorphism
--> Abstraction
--> Encapsulation

Rules for Naming:
	--> should not contain any spaces
	--> should not contain any symbols(except _ , $)
	--> should not start with numeric

Standard Notations:
====================
1)Pascal Notation:
------------------
	- Every words first letter should starts with upper case
	Eg:- GreensTechTrichy
	==> Project , Class

2)Camel Notation:
-----------------
	- First words first letter should be lower case and remaining words should starts with upper case
	Eg:- greensTechTrichy
	==> method , object , variable


Package	==> org.packagename / com.packagename
